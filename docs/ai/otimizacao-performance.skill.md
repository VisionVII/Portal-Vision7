# Skill: Otimização de Performance do Vision

## Descrição
Esta skill especializa o agente em otimizar performance do portal, garantindo carregamento rápido, responsividade e experiência fluida para usuários.

## Domínios de Aplicação
- Aplicações React/Vite
- Portais de conteúdo pesados
- Aplicações web públicas
- Mobile web performance

## Capacidades
### Frontend Performance
- **Bundle Optimization**: Code splitting, tree shaking
- **Asset Optimization**: Imagens, fonts, CSS/JS minification
- **Caching Strategies**: Browser caching, service workers
- **Lazy Loading**: Components e routes

### Database Performance
- **Query Optimization**: Efficient Supabase queries
- **Pagination**: Virtual scrolling para listas grandes
- **Data Fetching**: React Query optimization
- **Real-time Updates**: Supabase realtime quando necessário

### Network Optimization
- **CDN**: Asset delivery optimization
- **Compression**: Gzip, Brotli
- **Preloading**: Critical resources
- **Resource Hints**: DNS prefetch, preconnect

### Monitoring e Analytics
- **Performance Metrics**: Core Web Vitals
- **Error Tracking**: Sentry integration
- **User Experience**: Real User Monitoring
- **A/B Testing**: Performance experiments

## Como Usar Esta Skill
Ative quando trabalhar com:
- Otimização de carregamento inicial
- Melhorias de performance percebida
- Otimização de queries
- Implementação de caching
- Análise de bottlenecks

## Exemplos de Uso
- "Implementar code splitting no portal"
- "Otimizar queries do Supabase"
- "Configurar lazy loading de imagens"
- "Melhorar Core Web Vitals scores"
