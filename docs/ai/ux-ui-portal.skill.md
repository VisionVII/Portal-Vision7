# Skill: UX/UI do Vision

## Descrição
Esta skill especializa o agente em criar e otimizar interfaces de usuário excepcionais para o portal de notícias, focando em usabilidade, acessibilidade e performance.

## Domínios de Aplicação
- Portais de notícias e conteúdo
- Aplicações React com shadcn/ui
- Design systems responsivos
- Interfaces mobile-first

## Capacidades
### Design System
- **Componentes Consistentes**: Uso de shadcn/ui e Tailwind CSS
- **Paleta de Cores**: Tema Portugal com cores nacionais
- **Tipografia**: Fontes Playfair Display e Inter otimizadas
- **Espaçamento**: Sistema de espaçamento consistente

### UX/UI Melhores Práticas
- **Mobile-First**: Design responsivo prioritizando mobile
- **Navegação Intuitiva**: Menu, breadcrumbs, paginação
- **Conteúdo Hierárquico**: Destaques, cards, layouts de grid
- **Feedback Visual**: Estados de loading, sucesso, erro

### Responsividade e Performance
- **Breakpoints**: Otimização para todos os tamanhos de tela
- **Performance**: Lazy loading, otimização de imagens
- **Acessibilidade**: WCAG 2.1 AA compliance
- **SEO**: Meta tags, structured data

### Tecnologias Modernas
- **React 18**: Hooks, Suspense, Concurrent Features
- **TypeScript**: Type safety e melhor DX
- **Tailwind CSS**: Utility-first CSS
- **shadcn/ui**: Componentes acessíveis e customizáveis

## Como Usar Esta Skill
Ative quando trabalhar com:
- Novos componentes de UI
- Otimização de layouts
- Implementação de responsividade
- Melhorias de acessibilidade
- Design de novas páginas

## Exemplos de Uso
- "Criar componente de card de notícia responsivo"
- "Implementar navegação mobile-friendly"
- "Otimizar performance de carregamento de imagens"
- "Adicionar acessibilidade ao formulário de newsletter"
